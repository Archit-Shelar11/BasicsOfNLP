# 🧠 Natural Language Processing (NLP) Basics

This repository contains fundamental implementations of **Natural Language Processing (NLP)** techniques using Python. It covers essential preprocessing steps, feature extraction methods, and machine learning models for text classification tasks such as **sentiment analysis**.

---

## 🚀 Features

### 🔹 Text Preprocessing
- Lowercasing  
- Removing HTML tags  
- Removing stopwords  
- Tokenization  
- Stemming  
- Lemmatization  

### 🔹 Feature Extraction
- Bag of Words (BoW)  
- TF-IDF  
- Word2Vec  

### 🔹 Machine Learning Models
- Random Forest Classifier  
- Sentiment Analysis on text data  

---

## 🛠️ Tech Stack

- Python 🐍  
- NLTK  
- scikit-learn  
- NumPy  
- Pandas  
- spaCy  
- TextBlob
